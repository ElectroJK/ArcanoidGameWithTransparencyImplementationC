# ArcanoidGameWithTransparencyImplementationC

## Requirements

Before compiling, ensure you have the following installed:

- [MSYS2](https://www.msys2.org/)
- g++ (C++17 or newer)
- Make
- OpenGL development libraries
- GLFW
- GLAD

You can install dependencies in MSYS2 (64-bit MinGW environment) using:

```bash
pacman -S mingw-w64-x86_64-gcc mingw-w64-x86_64-glfw mingw-w64-x86_64-make
```

Build and Run
Open the MSYS2 MinGW64 terminal.
Navigate to the project source directory:
```bash
cd /c/Users/USER/Projectname/src
make
./ArcanoidTheGame.exe
```
